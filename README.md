# Cat-Mirror🐈
## Modular Smart Mirror System written using Flask + Electron

## Pre-requirements:
- Python3 🐍
- NPM + Node.JS ☕

## Installation
    npm install
    pip install -r requirements.txt

## Usage
Simply type `npm start` to boot the mirror.
